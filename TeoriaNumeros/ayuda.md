⭐ Regla 1 — Si el problema dice “potencia”, “inverso”, “mod”, “exponente enorme” → usar exponenciación binaria.

Ejemplo:
A1. Exponentiation.

⭐ Regla 2 — Si el problema dice “divisores”, “cantidad de divisores”, “factorizar” → usar SPF (Criba de Eratóstenes).

Ejemplo:
A2. Counting Divisors.

⭐ Regla 3 — Si el problema dice “combination”, “choose”, “nCk”, o “formas de elegir” → usar combinatoria (binomiales).

Ejemplo:
A3. Binomial Coefficients.
